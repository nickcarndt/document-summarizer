## Taking Screenshots

1. Run the app locally: `npm run dev`
2. Navigate through the app to capture each view
3. Use browser dev tools or a screenshot tool
4. Save screenshots with the exact filenames listed above
5. Ensure screenshots are clear and show the key features

## Notes

- Screenshots should be taken from the production or staging environment for best quality
- Ensure all UI elements are visible and not cut off
- Use consistent styling/theme across all screenshots
- Consider using a tool like CleanShot X, Shottr, or browser dev tools for high-quality captures
